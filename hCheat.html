HTML5 Cheat Sheet:

HTML5 Tag: Description: Syntax:
Paragraph Tag Let's user to create paragraphs. <p>
Header Tag Lets user to create headers(h1-h6). <h1>
Anchor Tag Lets users to creat external links. <a>
Divider Tag Lets users to devide their code. <div>
Subtitle Tag Lets users to create subtitles. <sub>
Image Tag Lets users to insert images. <img>
